# Smartphone Addiction Prediction using Machine Learning

## Project Overview

This project analyzes smartphone usage behavior and predicts the likelihood of smartphone addiction using machine learning techniques. The dataset contains behavioral, academic, and demographic information of users, which helps understand how smartphone usage impacts stress levels, academic performance, and addiction.

The goal of the project is to explore patterns in smartphone usage and build predictive models that can classify addiction levels.

---

## Dataset

The dataset contains **7500 rows and multiple behavioral features**, including:

* Gender
* Stress Level
* Academic Work Impact
* Daily Smartphone Usage
* Social Media Usage
* Addiction Level
* Addiction Label (Target Variable)

These features help analyze how smartphone habits affect users.

## Machine Learning Models Used

* Logistic Regression
* Decision Tree

Pipelines and column transformers were used to automate preprocessing and model training.
The Decision Tree model achieved 99% accuracy, while Logistic Regression achieved 88% accuracy on the test dataset. The Decision Tree performed better in capturing nonlinear relationships between smartphone usage features and addiction levels.
